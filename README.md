# Gestión Financiera

Una aplicación austera y directa para el control riguroso de las finanzas personales. 

El objetivo principal es proporcionar una herramienta funcional, sin distracciones, orientada a la administración del patrimonio personal.

## Características Principales

* **Registro de Movimientos:** Control exacto de ingresos y gastos.
* **Gestión de Presupuestos:** Asignación de límites financieros por categoría.
* **Metas de Ahorro:** Seguimiento del progreso hacia objetivos de capitalización.
* **Interfaz Funcional:** Diseño sobrio, enfocado en la claridad de los datos.
* **Personalización Básica:** Ajustes de perfil y temas (modo claro/oscuro y paletas de colores sobrias) adaptados para la legibilidad.
